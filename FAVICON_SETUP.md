# Favicon Setup Instructions

## Quick Setup (Recommended)

1. Save your BiteBook logo image as `icon.png` in the `src/app/` directory
2. Next.js will automatically generate all required favicon sizes
3. The favicon will appear on your website automatically

## File Location
- Place the logo file at: `/src/app/icon.png`

## Image Requirements
- Format: PNG (recommended) or SVG
- Size: At least 512x512 pixels (larger is better)
- Background: Transparent or white
- The logo should be centered and clearly visible at small sizes

## Alternative: Manual Setup

If you want to provide specific sizes, save these files in the `public/` directory:
- `favicon.ico` - Traditional favicon (16x16, 32x32, 48x48)
- `icon-16x16.png` - 16x16 pixels
- `icon-32x32.png` - 32x32 pixels  
- `apple-icon.png` - 180x180 pixels (for Apple devices)

## Online Tools for Conversion

You can use these tools to convert your logo to favicon formats:
- https://realfavicongenerator.net/
- https://favicon.io/
- https://www.favicon-generator.org/

Just upload your logo image and download the generated favicon files.
