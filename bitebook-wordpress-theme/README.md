## Installation Instructions

### 1. SiteGround Setup
1. **Purchase SiteGround Hosting**
   - Choose a plan (GrowBig or GoGeek recommended)
   - Select a domain or use existing domain

2. **Install WordPress**
   - Use SiteGround's WordPress installer
   - Choose latest WordPress version
   - Set up admin credentials

### 2. Upload Theme
1. **Download Theme Files**
   - Download the entire `bitebook-wordpress-theme` folder
   - Compress it into a ZIP file

2. **Upload via WordPress Admin**
   - Go to Appearance > Themes > Add New
   - Click "Upload Theme"
   - Choose the ZIP file and install
   - Activate the BiteBook theme

### 3. Configure Content
1. **Add Monthly Promotions**
   - Go to Monthly Promotions > Add New
   - Create entries for each month with:
     - Title (e.g., "Taco Month")
     - Tagline (e.g., "Love at First Bite")
     - Icon (emoji like ðŸŒ®)
     - Month selection
     - Color scheme
     - Description

2. **Add Partner Restaurants**
   - Go to Partner Restaurants > Add New
   - Add restaurant details:
     - Restaurant name
     - Location
     - Specialty
     - Rating (1-5)

3. **Customize Settings**
   - Go to Appearance > Customize
   - Configure Hero Section
   - Set pricing information
   - Update contact details

### 4. Create Pages
1. **Checkout Page**
   - Create new page titled "Checkout"
   - Set template to "Checkout Page"
   - Publish the page

2. **Additional Pages**
   - Terms of Service
   - Privacy Policy
   - Contact Us
   - Partner Application
