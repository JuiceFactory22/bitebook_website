1. Go to https://dashboard.emailjs.com/
2. Click **Email Templates** in the left sidebar
3. Find your lead magnet template (the one with the coupon content)
4. Click on it to open it

### Step 2: Get the Correct Template ID

1. In the template editor, go to the **Settings** tab
2. Look for **"Template ID"** field
3. Copy the EXACT Template ID shown there
4. It might be:
